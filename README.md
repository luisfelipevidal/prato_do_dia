#  PratoDoDia
Sistema de economia circular para reduzir o desperdício de comida.

##  O que é?
O PratoDoDia conecta padarias e restaurantes que têm comida sobrando a clientes que querem comprar essa comida com 50% a 70% de desconto.

##  Tecnologias Utilizadas
- HTML e CSS (Interface Minimalista)
- Node.js (Simulação de Back-End)
- Banco de Dados: MySQL (Tabelas: clientes, estabelecimentos, caixas_surpresa, pedidos)

##  Como testar
1. Baixe os arquivos da pasta `/frontend` e abra o `login.html` no seu navegador.
2. Importe o arquivo `schema.sql` da pasta `/database` no seu phpMyAdmin para criar o banco de dados.
